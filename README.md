# AI-Powered Intelligent Ordering System
**AI 智能点餐分析系统 | 大模型 + 数据分析 + 业务落地项目**


---

## 📌 项目介绍
本项目是一个**基于大模型与向量数据库的智能点餐分析系统**，通过 AI 能力实现菜品智能推荐、用户意图识别、配送范围分析、多源数据处理与业务决策支持。

项目特色：
- 融合 **LLM + 数据分析 + 业务系统** 全流程能力


---

## 🧠 技术栈
- 大模型框架：LangChain
- AI 模型：通义千问（Qwen）
- 后端服务：FastAPI
- 数据库：MySQL + Pinecone 向量数据库
- 地图服务：高德地图 API
- 开发语言：Python
- 部署：Uvicorn

---

## ✨ 核心功能
1. **智能对话与意图识别**
   - 自动识别用户问题类型（常规咨询 / 菜品推荐 / 配送查询）
   - 基于 LLM 做语义理解与智能回复

2. **菜品智能推荐系统**
   - 向量数据库存储菜品信息
   - 基于语义相似度做精准推荐
   - 结构化数据分析输出

3. **配送范围智能分析**
   - 地址解析、距离计算
   - 配送范围判断与策略分析
   - 支持步行/骑行/驾车三种模式

4. **完整数据处理链路**
   - MySQL 业务数据读取
   - 向量数据构建与检索
   - 数据格式化、清洗、结构化
   - 分析结果可视化输出
